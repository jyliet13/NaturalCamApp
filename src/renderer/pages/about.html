ACERCA DE LA APP

Esta aplicación de escritorio es parte de un Trabajo de Fin de Grado (TFG) dedicado a la clasificación de imágenes de fósiles utilizando técnicas de Inteligencia Artificial.

Desarrollada con **Electron** para la interfaz de usuario, **Python (Flask)** para el backend de la API y **TensorFlow/Keras** para el modelo de clasificación de imágenes.
